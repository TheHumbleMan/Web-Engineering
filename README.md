# Installation
## Vorraussetzungen

- Node.js (node -v)
- npm (npm -v)

## Abhängigkeiten installieren

- npm install
## Anwendung starten

- node app.js (aktuell eine Übergangslösung bis richtiges Backend existiert)

## Im Browser öffnen
```http:localhost:3000/auth/login```
oder je nach nginx Verknüfpung auch mit URL
